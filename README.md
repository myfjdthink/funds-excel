# funds-spy

## Description

将天天基金的排行数据转为 Excel 下载。

参考：
http://fund.eastmoney.com/data/fundranking.html
https://bbs.feidee.com/thread-585239-1-1.html

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
